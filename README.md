Overview

A Java-based web application using Servlets, JSP, and Oracle 11g XE to manage books and authors in a library.
Supports CRUD operations for authors and books with proper validation.

Features

Add, view, and manage authors
Add, view, and manage books
Validates author selection to prevent invalid entries
Uses Oracle 11g XE as the database
JSP and HTML front-end with success and failure pages

##Technologies Used

Java 11 / 17
Servlets & JSP
Oracle 11g XE
JDBC
HTML / CSS
Eclipse IDE / Tomcat 9.0
